# NextFlix

É um projeto de recriação da página inicial da Netflix desenvolvido por mim com propósito de evoluir os estudos em NextJS & consumo de API.

# Tecnologias

- NextJS
- TypeScript
- Styled Components
- Axios
- API do TMDB
